# CodePeer

CodePeer is an advanced, AI-driven educational platform designed to provide a comprehensive Data Structures and Algorithms (DSA) learning environment. It moves beyond simple "correct/incorrect" feedback by acting as a Socratic mentor, encouraging active problem-solving and deeper understanding.

## Features

*   **Expansive Problem Library:** Over 70 curated DSA problems ranging from Easy to Hard, covering topics like Arrays, Strings, DP, Graphs, and more.
*   **Real-Time Code Linting:** Immediate feedback on syntax errors and common mistakes (e.g., missing colons, unmatched brackets) as you type, complete with visual error markers.
*   **Robust Test Runner:** JavaScript solutions are validated against comprehensive test cases using deep comparison. Python and C++ solutions receive structural analysis and guidance for local testing.
*   **Socratic Mentor AI:** Instead of giving you the answer, the platform analyzes your code and provides contextual, thought-provoking questions to guide you towards the solution.
*   **Rubber Duck Mode (Speech-to-Text):** Stuck? Explain your approach out loud! The AI analyzes your verbal reasoning and, if your logic is sound, unlocks advanced "Clue" hints.
*   **Time-Travel Refactor Challenge:** Solving a problem is just the first step. The platform challenges you to refactor your code to meet stricter time or space complexities (e.g., from $O(n^2)$ to $O(n \log n)$).
*   **Skill Tree Dashboard:** An RPG-style dashboard that tracks your progress across key competencies like Time Complexity, Data Structures, and Edge Case Handling over your session.
*   **Dynamic Visualizations:** Generates flowcharts for loops and recursive functions to help visualize execution flow.

## Getting Started

Simply open `index.html` in your web browser to start coding. The application runs entirely client-side, utilizing standard web technologies (HTML, CSS, JavaScript) and CodeMirror 6 for the editing experience.

## Technology Stack

*   HTML5 / Vanilla CSS
*   Vanilla JavaScript
*   CodeMirror 6 (Code Editor)
*   Mermaid.js (Flowchart generation)
*   Web Speech API (Rubber Duck Mode)
